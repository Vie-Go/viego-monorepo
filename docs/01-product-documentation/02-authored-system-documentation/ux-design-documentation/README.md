## Experience principles
- **Crimson & gold anchor** — primary crimson `#BE382A` for actions, gold `#F2B72F` for streaks
  and unlocked territory.
- **Organic tactility** — rounded forms, ambient shadows, rewarding unlock/streak micro-animations.
- **Mobile-first rhythm** — compact single-handed layout, ≥44px touch targets.
- **Dual-mode harmony** — seamless light/dark.
